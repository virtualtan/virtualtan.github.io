工具推荐——好用的网站
2020-06-08|日志
Word count: 678|Reading time: 2 min

本文写了博主常用的网站，学习娱乐都有，可以收藏一波…

ps：对于娱乐性的网站，同类型的只推荐一种，有兴趣的可以直接去我的导航，那里收录了大部分网站。

自主学习类

  1. 仿-知网

    类似知网，但无广告，更好用，写论文的可以收藏下。

  2. TED

    知名英文演讲平台，可以学到很多知识，还能练习英语，两开花。

  3. 网易公开课

    知名公开课平台，也有TED的视频（配中英字幕），还有其他的知识，比如知名大学的公开课之类的，推荐一下。

  4. 慕课MOOC

    超级有名的公开课平台，各类学习视频都有，学习范围广泛，付费免费资源很多，推荐一下。

  5. w3school

    自学计算机语言相关知识，很不错的网站，免费，推荐一下。

  6. 菜鸟教程

    和上面的 5 差不多。

  7. coursera

    被称为外国的慕课，我没经常去，有兴趣的可以去看看。

附：相关电子书籍资源，下面给出几个网站，可能会帮到您

鸠摩搜书, 雅书, 苦瓜书盘, 计算机书籍

建议支持正版

优质社区

  1. 少数派

    有很多有价值观看的文章，教程，还有软件推荐，科技报。

  2. 百度贴吧

    还是有优质帖子的….这个就自己去看吧

  3. 知乎

    有很多大牛，大部分是真才实学，言论有理有据，可以学到东西

  4. V2EX

    知名网络问答社区，很不错

  5. github

    全球最大同性交友网站， 开源网站，全球知名

软件下载

MacWK

精品Mac软件下载网站，美观，好用，无毒，友好。

其他下载网站详见导航：优质软件探索

影音-漫画-小说

  1. 影音：

    片库 : 可下载，可高速在线观看，可解析

    bilibili： 这个不多bb，都知道

    全网音乐下载: 好用，有条件的支持正版

  2. 漫画

    漫画堆：更新漫画相对较快，更快的可以自行百度

  3. sodu搜小说： 搜小说，不能下载，但小说更新超快，要下载的自己找网站吧

其他的看电影，看什么的网站详见导航：影音-漫画-小说

  2. veryins: 国内收录ins精品图片的网站
  3. sm.ms: 图床
